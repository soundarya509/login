Installation
1. Clone the Repository
git clone https://github.com/soundarya509/login.git
2. Navigate to Project
cd login-app
3. Install Backend Dependencies
cd backend
npm install
4. Start Backend Server
node server.js

Server will run on:

http://localhost:5000
5. Run Frontend

Open the frontend/index.html file in your browser.

Remember Username Feature

After a successful login, the username is stored using localStorage.
The next time the user opens the application, the username will automatically appear in the login field.
